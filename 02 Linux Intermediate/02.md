## 2. The File System

### 2.9 Disk Usage
#### 2.9.1 `df`
The `df` command shows you the utilization of your currently mounted filesystems. 

- `-a`, `--all`: 包含所有的具有 0 Blocks 的文件系统
- `-h`, `--human-readable`: 使用人类可读的格式
- `-i`, `--inodes`: 列出 inode 资讯，不列出已使用 block
